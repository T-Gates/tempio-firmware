# 🗺️ 빵판 연습 로드맵 — 서늘 허브 준비

ESP32-C3 SuperMini로 빵판 실습. 각 단계가 빵판 실력 + 서늘 프로젝트 부품을 동시에 익히도록 구성.
**한 빵판 위에 단계마다 부품을 "추가"하며 누적** — 7단계에서 허브 하드웨어 골격 완성.

---

## 1단계 — LED 마스터 ✅
- [x] LED 1개 깜빡
- [x] LED 2개 저항 분리해서 병렬 연결
- [x] LED 2개를 다른 핀(GPIO3·GPIO4)에 → 번갈아 깜빡
- [x] LED 3개로 신호등 (순차 점등)

배우는 것: 빵판 행/열/홈 구조, 멀티 출력 제어, 점퍼선 정리
부품: LED 3개, 저항(220~330Ω), 점퍼선

## 2단계 — 버튼 입력 ✅
- [x] 택트 스위치 + INPUT_PULLUP 으로 버튼 상태 읽기
- [x] 버튼 누르면 LED 켜기
- [x] 누를 때마다 토글 (디바운싱 포함)

배우는 것: 디지털 입력, 풀업 저항, 입력+출력 한 빵판에
부품: 택트 스위치 1~2개

## 3단계 — PWM 밝기 조절 ✅
- [x] analogWrite로 LED 밝기 단계별 조절
- [x] LED 페이드 (숨쉬는 효과)

배우는 것: PWM — 디지털로 아날로그 흉내
부품: 기존 것

## 4단계 — 아날로그 입력 ★서늘 LDR 직결
- [ ] 가변저항으로 analogRead → 시리얼로 값 확인
- [ ] LDR + 저항 분압회로 구성
- [ ] 어두워지면 LED 자동 켜기

배우는 것: ADC, 분압회로(voltage divider) — 서늘 허브 조도센서
부품: 가변저항, LDR, 10kΩ 저항

## 5단계 — DHT22 온습도 ★서늘 허브 센서 ✅
- [x] DHT 라이브러리 설치 (lib_deps)
- [x] 온습도 읽어 시리얼 출력
- [x] 측정값 주기적으로 갱신

배우는 것: 디지털 센서 프로토콜, 라이브러리 관리
부품: DHT22 (보유 중)

## 6단계 — 트랜지스터 스위칭 ★서늘 IR노드 직결
- [ ] NPN 트랜지스터로 LED 켜기
- [ ] GPIO 신호로 트랜지스터 스위칭 이해

배우는 것: 트랜지스터 스위치 — IR노드가 IR LED 구동하는 원리
부품: NPN 트랜지스터(2N2222 등), IR LED 또는 일반 LED

## 7단계 — 허브 프로토타입 통합 🎯
- [ ] 빵판 하나에 ESP32-C3 + DHT22 + LDR + 상태 LED 전부 배선
- [ ] 전원 레일 공유, 배선 정리
- [ ] 센서값 모두 시리얼로 동시 출력

배우는 것: 멀티 컴포넌트 배선 = 서늘 허브 하드웨어 골격
→ 이후 WiFi·HTTP 얹으면 진짜 허브
