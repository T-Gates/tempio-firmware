# tempio 펌웨어 개발 로드맵

각 단계가 독립적으로 플래시·검증 가능하도록 구성. 이전 단계 코드를 그대로 확장해서 올라감.

---

## Phase 1 — BLE 연결 (C3 두 대)

ESP32-C3 2대로 BLE Central↔Peripheral 연결 확인. 하드웨어 센서 없이 순수 BLE만.

- [ ] Peripheral: 광고(advertise) + 서비스/특성 등록 + mock 센서 데이터 전송
- [ ] Central: 스캔 → 연결 → 데이터 읽기 → 시리얼 출력
- [ ] 양방향 확인: Central→Peripheral 명령 전달 (LED 토글 등)

검증: 시리얼 모니터 2개 열어서 양쪽 로그 확인

## Phase 2 — 센서노드 기본

센서노드 보드(ESP32-C3)에 SHT40 + LDR 연결, BLE로 허브에 전송.

- [ ] SHT40 I2C 연결 + 온습도 읽기
- [ ] LDR ADC 읽기 (GPIO2)
- [ ] BLE Peripheral로 실제 센서값 전송 (Phase 1 코드 확장)
- [ ] 배터리 전압 ADC 읽기

검증: Central 쪽 시리얼에서 실제 온습도·조도 값 수신 확인

## Phase 3 — 허브 WiFi + 서버 연동

허브 보드(ESP32-S3)에서 WiFi + HTTP POST. BLE Central 기능과 통합.

- [ ] ESP32-S3로 Central 코드 포팅 (C3→S3 차이 처리)
- [ ] WiFi 연결 + HTTP POST `/api/hub/{hub_id}/report`
- [ ] BLE로 받은 센서노드 데이터를 서버에 업로드
- [ ] 서버 응답의 `commands` 배열 파싱

검증: 테스트 서버 대시보드에서 실시간 데이터 확인

## Phase 4 — 허브 SCD40 (CO2)

허브에 SCD40 센서 연결, CO2 + 기준 온습도 측정.

- [ ] SCD40 I2C 연결 + CO2/온습도 읽기
- [ ] 서버 리포트에 CO2 데이터 포함
- [ ] 센서노드 데이터와 허브 온습도 교차 출력

검증: 대시보드에서 CO2 값 + 센서노드 온습도 동시 표시

## Phase 5 — IR노드 기본

IR노드(ESP32-C3) BLE Peripheral + IR 발사.

- [ ] BLE Peripheral로 허브에서 IR 타이밍 수신
- [ ] 38kHz PWM 캐리어 생성 (ledcWrite)
- [ ] 수신한 타이밍 배열대로 IR LED 발사
- [ ] 발사 결과 보고 (성공/실패)

검증: VS1838B IR 수신기로 신호 확인, 또는 실제 에어컨 반응 확인

## Phase 6 — 서버 능동 제어

피기백(허브가 POST할 때 응답에 명령 실어보내기)만으로 충분한지, 서버→허브 push가 필요한지 검증.

- [ ] 피기백 지연 측정: 서버에서 명령 생성 → 허브가 다음 POST로 수신까지 실제 지연 시간
- [ ] WebSocket 테스트: 서버→허브 즉시 push (HTTP 기반, 추가 인프라 불필요)
- [ ] MQTT 테스트: 브로커(Mosquitto) 띄우고 서버→허브 pub/sub
- [ ] 판정: 피기백으로 충분하면 유지, 즉시 제어가 필요하면 WebSocket 또는 MQTT 채택

검증: 서버 대시보드에서 "26도 냉방" 버튼 누르기 → IR노드가 에어컨에 IR 발사까지 E2E 지연 측정

## Phase 7 — 딥슬립

센서노드·IR노드에 딥슬립 적용. 배터리 구동 준비.

- [ ] 센서노드: 1분 딥슬립 → wake-up → BLE 연결 → 데이터 전송 → 슬립
- [ ] IR노드: 5분 딥슬립 → wake-up → BLE 연결 → 명령 확인 → 슬립
- [ ] 허브: 노드 wake-up 시 자동 재연결 처리
- [ ] 전류 측정 (멀티미터로 슬립 중 전류 확인)

검증: 24시간 연속 동작 + 배터리 전압 변화 추적

## Phase 8 — 통합 테스트

전 노드 동시 운영. 실제 매장 배치 전 최종 검증.

- [ ] 허브 1 + 센서노드 2 + IR노드 1 동시 운영
- [ ] 서버에서 IR 명령 → 허브 → IR노드 → 에어컨 동작 E2E
- [ ] 24시간 안정성 테스트 (연결 끊김, 재연결, 메모리 누수)
- [ ] 배터리 수명 실측 시작
